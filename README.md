## 🖥️ How to Mount an ISO File

Mounting a file makes your computer think you inserted a physical disc. This is useful for running games or software installers that require a disc.

1.  Locate the .iso file on your computer.
2.  Right-click the file.
3.  Look for the MagicISO-Premium option in the menu.
4.  Select the "Mount Image" command.
5.  Open "This PC" in your File Explorer.
6.  You will see a new virtual disc drive listed under "Devices and drives."
7.  Double-click this new drive to access the contents of your file.
